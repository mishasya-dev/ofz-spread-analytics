# Будущие фичи (Future Features)

Файлы и модули, которые содержат неиспользуемый код, но могут быть полезны в будущем.

## 📁 modes/ - Режимы работы

### Статус: 🟡 Не используется (v0.2.3)

**Файлы:**
- `modes/base.py` - Базовый класс BaseMode
- `modes/intraday.py` - Класс IntradayMode

**Описание:**
Изначально проектировались как архитектура для разделения логики дневного и внутридневного режимов. В текущей версии v0.2.x логика реализована напрямую в `app.py` и компонентах.

**Планируется:**
- Рефакторинг для использования классов режимов
- Это позволит лучше тестировать и расширять функциональность

---

## 📁 core/backtest.py - Бэктестинг

### Статус: 🟡 Не используется (v0.2.3)

**Описание:**
Модуль для тестирования торговых стратегий на исторических данных.

**Функциональность:**
- BacktestPosition - позиция в бэктесте
- BacktestResult - результаты бэктеста
- backtest_strategy() - запуск бэктеста

**Планируется:**
- Интеграция с UI в версии v0.3.0
- Графики результатов бэктеста (уже есть в charts.py)

---

## 📁 export/signal_sender.py - Отправка сигналов

### Статус: 🟡 Не используется (v0.2.3)

**Описание:**
Модуль для отправки торговых сигналов через различные каналы.

**Функциональность:**
- SignalSender - базовый класс
- TelegramSignalSender - отправка в Telegram
- WebhookSignalSender - отправка на webhook

**Планируется:**
- Интеграция в версии v0.3.0
- Настройка через config.py (ExportConfig уже определён)
