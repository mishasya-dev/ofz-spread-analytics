## 4. Проблемы архитектуры

### 4.1. Два уровня БД
- **`core/database.py`**: старый модуль с `DatabaseManager`
- **`core/db/`**: новая архитектура с репозиториями

Оба используются! `app.py` импортирует `get_db` из `core.database`, а ZCYC использует `get_g_spread_repo` из `core.db`.

### 4.2. Экспорт в `__init__.py` не соответствует использованию

**`/__init__.py` экспортирует:**
```python
"TradingChecker", "TradingStatus",  # api - используется только в modes/
"HistoryFetcher", "CandleFetcher",   # api - используется
"YTMCalculator", "SpreadCalculator", # core - SpreadCalculator НЕ используется в app.py
"SignalGenerator", "TradingSignal",  # core - НЕ используется в app.py
"Backtester",                        # core - НЕ используется
"SignalSender", ...                  # export - НЕ используется
"DailyMode", "IntradayMode"          # modes - НЕ используется в app.py
```
