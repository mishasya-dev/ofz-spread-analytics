## 🚀 Последние оптимизации (16.03.2026)

### Trading Calendar

**Проблема:** При перезапуске приложения происходили лишние запросы к MOEX API из-за неверного определения выходных дней.

**Решение:** Реализован торговый календарь MOEX с загрузкой праздников и переносов.

| Компонент | Файл | Назначение |
|-----------|------|------------|
| CalendarRepository | `core/db/calendar_repo.py` | Сохранение/загрузка торговых дней |
| TradingCalendar | `services/trading_calendar.py` | Singleton с кэшем в памяти |
| Таблицы БД | `trading_calendar`, `calendar_meta` | Персистентность |

**Результат:**
- Система не запрашивает данные за выходные
- Предотвращены ложные триггеры полной перезагрузки

### ZCYC Loading Optimization (5x speedup)

**Проблема:** ZCYC загружался отдельно для каждой облигации = N × D запросов к MOEX

**Решение:** MOEX API возвращает ВСЕ 32 облигации за один запрос!
